# 協力会社フォーム自動送信システム

## ファイル構成
```
form_sender/
├── app.py              # Webアプリ本体（Flask）
├── engine.py           # フォーム自動送信エンジン
├── requirements.txt    # 必要なライブラリ
├── render.yaml         # Renderデプロイ設定
└── templates/
    └── index.html      # 画面UI
```

## セットアップ手順

### 1. GitHubにアップロード
1. GitHubアカウントを作成（無料）
2. 新しいリポジトリを作成
3. このフォルダをまるごとアップロード

### 2. Render.comにデプロイ
1. https://render.com でアカウント作成（無料）
2. 「New Web Service」をクリック
3. GitHubリポジトリを選択
4. 環境変数に ANTHROPIC_API_KEY を設定
5. デプロイ完了 → URLが発行される

### 3. 使い方
1. 発行されたURLをブラウザで開く
2. 「協力会社リスト.xlsx」をアップロード
3. テストモードで動作確認
4. 問題なければ本番モードで実行
5. 結果Excelをダウンロード

## Excelフォーマット
「協力会社フォーム送信管理.xlsx」を使用してください。
- シート1「協力会社リスト」: 送信者情報と協力会社リストを入力
- シート2「実行結果」: システムが自動で書き込み
- シート3「手動対応リスト」: 自動送信できなかった会社のみ抽出

## 注意事項
- CAPTCHAがあるサイトは自動送信できません（手動対応リストに出力）
- 営業禁止の記載があるサイトは自動でスキップします
- 必ずテストモードで動作確認してから本番実行してください
